# Design Spec — Guías de usuario Hydra Framework

**Fecha:** 2026-05-14  
**Tipo:** Documentación HTML estática  
**Archivos:** `docs/guide-dev.html`, `docs/guide-founder.html`

## Objetivo
Dos documentos HTML independientes que explican cómo usar Hydra Framework según el perfil del lector.

## guide-dev.html — Perfil técnico
- Tema: dark (igual que mission control)
- 7 pasos numerados con comandos copiables
- Paso 1: Requisitos | 2: Clonar+.env | 3: Docker | 4: inject.sh | 5: runner.sh | 6: health_check() | 7: ADAPTAR A

## guide-founder.html — Perfil founder
- Tema: light, limpio, emoji
- 6 pasos con analogías, sin código
- Paso 1: ¿Qué es? | 2: ¿Qué necesitas? | 3: Arrancar | 4: Elegir módulos | 5: Qué tienes el día 1 | 6: Siguiente paso

## Aprobado por usuario: sí
